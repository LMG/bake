How to install bake command line tool

bake is a ruby gem. It runs with ruby 1.8.6 and above. However, it is highly recommended, that you use ruby 1.9.x, because of a significant speed improvement.

bake and its depedencies can be found on rubygems, which is configured as ruby source per default. Type "gem sources" if you wish to check that.
Installing bake is very easy: 

gem install bake-toolkit

Problems

  • If the installation does not start, it might be a problem of a password protected internet proxy. Unlock the proxy by opening an external webpage e.g. google.
  • Note, that the error parsers in bake assume English language. It depends on the system and compiler how to switch the language. One possibility is to set the environment variable LC_ALL: 
    LC_ALL=en_US
  • Don't worry if you get an error like this: 
    ERROR:  While generating documentation for ...
... MESSAGE:   Error while evaluating ...
   undefined method `gsub' for nil:NilClass ...
    It's a ruby bug. However, the gems were installed correctly. You can avoid building the documentation by installing bake like this: 
     gem install bake-toolkit --no-rdoc --no-ri
