How to install bake Visual Studio integration¶

VS 2012¶

The plugin is available on a E.S.R.Labs update site. Please follow the steps below to install the plugin.

  1. Configure the update site

    http://www.esrlabs.com/vs-plugins/bake/atom.xml

  2. Now it’s available via Tools->Extensions and Updates

  3. Check if it’s installed correctly

  4. If an update is available, it will look like this

VS 2010¶

The plugin is available on a E.S.R.Labs server. Download Visual Studio 2010 bake plugin, double click the file and install it.
