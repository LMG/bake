How to install bake Eclipse integration¶

The bake Eclipse integration plugin is available on a E.S.R.Labs update site. Please follow the steps below to install the plugin.

  1. Go to “Help” -> “Install New Software”

  2. Setup the update site

    https://assets.esrlabs.com/eclipse-plugins/com.esrlabs.eclipse.bake.updatesite/

  3. Select the bake Feature

  4. Accept the license after reading it

  5. Ignore the security warning - you can trust the software

  6. After successfull installation, restart Eclipse
